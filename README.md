# cailloux
Cailloux is a utility to manage my hyprland configuration using stylus _signs_.
It is not meant to work on any computer and is tailored to be used on my personal computer
(a Samsung Galaxy Book5 360).

Still, feel free to use this software as you wish, fork it if you want to use it
for your personal usage or create a pull request if you have any useful contribution.

## Note

Your user has to be in the `input` group for this program to be able to
read the stylus.
